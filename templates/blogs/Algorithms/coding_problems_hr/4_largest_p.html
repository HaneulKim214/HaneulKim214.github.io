Solutions

Again, greedy approach will be taken to solve this problem. There are two ways of doing it.

  1. Get all possible array and take most optimal
  2. Have base array, update it each time more optimal array arise.

#1 Try
  • Only trying for k = 1
  • given an array, I find highest number in an array and swap it with the first element. Since having largest value as first element will give highest possible lexicographical value. 
def largestPermutation(k, arr):
    for i_swap in range(k):
        high_pos = arr.index(max(arr))
        arr[0], arr[high_pos] = arr[high_pos], arr[0] 
    return arr
  • This passes all test case but it won't work for k > 1
  • as k increase we must increase 0th position to 1st position and so on... But note that we should not consider already swapped value.

#2 Try
  • as we loop thorugh high_pos conders arr[i:] therefore ignoring already swapped value
  • find position of higest value from arr[i:] swap it with ith position. Note that previous values in array are swapped already.
def largestPermutation(k, arr):
    for i in range(k):
        high_pos = arr.index(max(arr[i:]))
        arr[i], arr[high_pos] = arr[high_pos], arr[i] 
    return arr

ex: if I pass in k = 2 , arr = [1,1,1,2,2,2]

  • first loop it correctly swaps giving me [2,1,1,1,2,2].
  • second loop I am looking for max value in [1,1,1,2,2] which should give an index value of 3 again. It gives high_pos = 0 instead. Giving arr = [1,2,1,1,2,2]


#3 Try
  • previously in 3rd line we are taking arr.index(highest_value) => after finding highest value in arr[i:] we find index in arr[:] in second loop we found 2 to be highest value but we look for its index in arr[:] which will ofcourse give us first position since we swapped it. So it should be arr[i:].index(highest_value) + i, add i at the end because we will swap values in original arr[:] thus must add value that has been cut off.
  • Also if k > len(arr) it outputted error because arr[k:] wouldn't exist if k > len(arr), so fixed that as well.
def largestPermutation(k, arr):
    if k <= len(arr):
        num_swaps = k
    else:
        num_swaps = len(arr)
    for i in range(num_swaps):
        high_pos = arr[i:].index(max(arr[i:])) + i
        arr[i], arr[high_pos] = arr[high_pos], arr[i] 
    return arr
  • Passes more test case but still fails on 7 of them.

What is LCS?

LCS stands for Longest Common Subsequence. First two words are pretty intuitive therefore let me explain what subsequence is.
We know that substring is a continous subset of a string. for example if we have a str = "hello" its substring would be "hel", "llo", and much more but not "hlo" since the letters are not continuous.

Subsequence is similar to substring however it does not have to be continuos yet still have to be in same relative order.
"hlo" would be subsequence of word "hello" however "lhe" is not since "h" must come first in the word "hello".
So longest common subsequence is in set of subsequences, one that has most common letters.

Let's further understand this concept following a simple example: 

s1 = "simple"
s2 = "examples"
subsequences = [{m,p,l,e}, {p,l,e}, {l,e}, {e}, {s}]
